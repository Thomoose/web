Protect Windows For Free  

Free security software is created to make money. The free versions work well but it's important to understand te difference between update and upgrade. Make sure to update the software to be able to prevent new threats but the software company want you to upgrade to the paid version that has more features.

The Basics
Windows is made by Microsoft. Windows versions 7, 8 and 10 are currently supported by Microsoft and security software vendors. If you have an older version of Windows (see the extensive list here), get ready to buy a new computer.

New computers usually come with a free trial version of anti-virus software and you're not protected when it expires. Trial software is not free. If you're not going to pay for it, uninstall it and install a highly functional free program.
IMPORTANT: Choose only one anti-virus program because they can conflict with each and cause system problems.

Microsoft provides good security software for free. For Windows 7, download Microsoft Security Essentials. Windows 8 and 10 have Windows Defender built in.

Here are three more good free antivirus scanners but there are many more.

  • Avast
  • AVG
  • BitDefender Free

There are many types of threats, viruses, malware, system and software exploits, rootkits, ransomware, and more. The free security software works well but they're generally focused on viruses.

For an additional level of protection I install Malwarebytes Anti-Malware. It's designed to work with anti-virus programs to catch other types of threats. The free version requires that you run the scanner manually to check your computer. The paid version runs in the background to prevent malware from getting into your computer.

If free sound like too much work, go for one of the Internet Security Suites. They combine the power of a free antivirus with Malwarebytes Anti-Malware in a single product.
